
# Career Map AI

A smart career transition platform that uses Artificial Intelligence to generate step-by-step roadmaps for professional growth.

## Tech Stack
- **Framework:** Next.js 15 (App Router)
- **AI:** Google Gemini 2.5 Flash API
- **Backend:** Supabase (Database & Auth)
- **Styling:** Tailwind CSS & Shadcn/UI

## Key Features
- **Social Auth:** Secure login with Google and GitHub.
- **AI-Powered Roadmaps:** Instant generation of career paths based on user input.
- **History Tracking:** Save, view, and delete previous roadmaps.
- **Responsive Design:** Optimized for both mobile and desktop.

## ⚙️ Local Setup
1. Clone the repo: `git clone https://github.com/ibadkath/career-path-ai.git`
2. Install dependencies: `npm install`
3. Set up your `.env.local` with your Supabase and Google AI keys.
4. Run the development server: `npm run dev`
